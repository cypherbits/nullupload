3. Terms

Uploading is showing your acceptance of the terms and conditions and can therefore be held responsible for illegal and/or copyright infringement material. Uploading means your acceptance of our privacy policy too.

4. Offenses

Do not upload anything that is illegal (child porn is not allowed). All illegal files will be removed as soon as possible when reported (we do not scan actively uploaded files). You can report files on our email: nullupload@protonmail.com or nullupload@tuta.io

5. Misuse

Files with the intention to cause harm to either the website, server, or users will be removed. Uploading files means you share these files and these files will be downloaded many times, using the service for personal long time backups where downloads are just a few is not permitted.

6. Report files

Any user have the right and must report illegal files to be deleted.
To report a file you send an email to nullupload@protonmail.com or nullupload@tuta.io with the download link of the offending file, if the content is protected with a password (eg. rar files) please, provide a password. We will read the report within 72 hours and review the content and delete the file if needed.

7. Time file removal

  • Files older than 1 day with less than 2 downloads will be removed
  • Files which last download date is older than 93 days will be removed

Optimizations

  • If a duplicated file (a file with the same content as another file already uploaded) is uploaded, we will delete the first uploaded file if the number of downloads is 0 and reemplace it with the new upload, if there are downloads, we won't allow the upload of the last file.
  • When the server disk is full we will select some random files to be removed based on the last download date and number of downloads
